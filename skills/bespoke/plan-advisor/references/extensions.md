# Extensions

Future capabilities that could extend plan-advisor.

## Planned

| Extension | Description |
|-----------|-------------|
| Multi-plan tracking | Advise on multiple plans simultaneously |
| Answer history | Persist auto-answer decisions for review |
| Custom quality profiles | Let users define project-specific quality criteria |
| Confidence scoring | Show confidence level with each auto-answer |

## Potential

| Extension | Description |
|-----------|-------------|
| Integration with implementation-plan skill | Auto-activate when implementation-plan creates a plan |
| Progress tracking | Monitor implementation progress against the plan |
| Risk escalation | Surface high-risk decisions to the user instead of auto-answering |
| Team conventions file | Load project-specific conventions to inform answers |
