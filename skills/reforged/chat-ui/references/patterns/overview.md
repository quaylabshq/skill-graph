# Patterns Overview

Proven usage patterns for building chat interfaces with chat-ui components.

## Pattern Map

| Pattern | When to Use | Reference |
|---------|-------------|-----------|
| Full Agent Assembly | Building a complete agent chat from scratch | [full-agent.md](full-agent.md) |
| Customization & Styling | Custom message rendering, theming, role-based styling | [customization.md](customization.md) |

## Choosing a Pattern

- **Starting from scratch?** → [Full Agent Assembly](full-agent.md) gives you the complete provider → layout → messages → input stack
- **Customizing existing chat?** → [Customization](customization.md) covers render props, markdown overrides, and Tailwind theming
